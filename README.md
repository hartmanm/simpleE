# simpleE

simpleE the inconspicuous ETH/USD ticker.  

Provides the exchange rate without distracting you.

![inthewildsoftheOS](https://github.com/hartmanm/simpleE/blob/main/simpleEv2.jpg)

## simpleE was developed using:

     Electron

     Node.js

     HTML5

## Platforms

     This version is for Windows

## How To Use

     Clone or Download the repository

          run simpleE

## Settings

     to move:  hold left click and drag

     to close:  right click and select close
